## Business Description

- **Business Description**: AWS Summit 2026 부스에서 AI-DLC(AI-Driven Development Life Cycle) 방법론을 시연하기 위한 인터랙티브 데모 웹 애플리케이션. 방문객이 만들고 싶은 서비스 아이디어를 입력하면, Kiro IDE 환경을 시뮬레이션하여 AI-DLC의 INCEPTION -> CONSTRUCTION -> OPERATIONS 전체 과정을 시각적으로 보여주고, 최종적으로 MVP UI, AWS 아키텍처, 프로젝트 예상 정보를 제공한다.

- **Business Transactions**:
  1. **데모 시작 (Demo Start)**: 방문객이 프로젝트 아이디어를 입력하고 데모를 시작하는 트랜잭션
  2. **AI-DLC 단계 진행 (Step Progression)**: 7단계의 AI-DLC 과정(요구사항 분석, 사용자 스토리, 애플리케이션 설계, NFR 설계, 코드 생성, 인프라 설계, 배포 계획)을 순차적으로 시뮬레이션
  3. **결과 조회 (Result View)**: MVP 미리보기, AWS 아키텍처, 비즈니스 워크플로우, 프로젝트 예상, AI-DLC 산출물, Kiro 소개를 탭 형태로 조회
  4. **이메일 리포트 전송 (Email Report)**: 데모 결과를 이메일로 전송 요청
  5. **세션 로깅 (Session Logging)**: 데모 시작/완료 이벤트를 서버에 기록

- **Business Dictionary**:
  | 용어 | 의미 |
  |------|------|
  | AI-DLC | AI-Driven Development Life Cycle. AI 기반 소프트웨어 개발 생명주기 방법론 |
  | INCEPTION | 요구사항 분석, 설계 등 계획 단계 |
  | CONSTRUCTION | 코드 생성, NFR 설계 등 구현 단계 |
  | OPERATIONS | 인프라 설계, 배포 계획 등 운영 단계 |
  | Kiro | AWS의 AI 기반 통합 개발 환경 (IDE) |
  | MVP | Minimum Viable Product. 최소 기능 제품 |
  | 산업 시나리오 | 이커머스, 핀테크, 헬스케어, 교육, 물류, SaaS, 채팅 등 산업별 분기 로직 |
  | 페르소나 | MVP 미리보기에서 사용자/관리자 두 가지 관점 |

## Component Level Business Descriptions

### StartPage (시작 페이지)
- **Purpose**: 방문객이 프로젝트 아이디어를 입력하고 데모를 시작하는 진입점
- **Responsibilities**: 아이디어 입력 폼 제공, 6개 산업 카테고리별 예시 제공, 세션 초기화

### DemoPage (데모 진행 페이지)
- **Purpose**: Kiro IDE 환경을 시뮬레이션하여 AI-DLC 7단계를 시각적으로 보여줌
- **Responsibilities**: 산업별 시나리오 감지, 단계별 파일 생성/채팅 메시지 시뮬레이션, 수동 단계 진행 제어

### ResultPage (결과 페이지)
- **Purpose**: AI-DLC 완료 후 종합 결과를 6개 탭으로 제공
- **Responsibilities**: MVP 미리보기, AWS 아키텍처 다이어그램, 비즈니스 워크플로우, 프로젝트 예상, AI-DLC 산출물, Kiro 소개, 이메일 리포트 전송

### EstimateService (예상 계산 서비스)
- **Purpose**: 프로젝트 아이디어의 키워드를 분석하여 개발 기간, 팀 구성, 비용을 동적으로 산출
- **Responsibilities**: 복잡도 분석, 개발 일수 계산, AI 절감 효과 산출, 팀 구성 추천, 비용 예상
