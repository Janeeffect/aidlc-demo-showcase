# User Personas

## Persona 1: 방문객 (Summit Visitor)

| 항목 | 내용 |
|------|------|
| **이름** | 김서연 (대표 페르소나) |
| **역할** | AWS Summit 2026 방문객 |
| **배경** | 스타트업 CTO / 개발팀 리드 / 기술 관심 일반인 등 다양한 기술 수준 |
| **목표** | AI-DLC 방법론이 실제로 어떻게 동작하는지 직접 체험하고, 자신의 프로젝트 아이디어로 MVP가 만들어지는 과정을 확인 |
| **동기** | AI 기반 개발 도구의 가능성 탐색, Kiro IDE에 대한 관심, 자사 프로젝트에 적용 가능성 평가 |
| **기술 수준** | 초급~고급 (다양) |
| **사용 환경** | AWS Summit 부스의 태블릿/데스크톱, 안정적 네트워크 |
| **언어** | 한국어 또는 영어 |
| **Pain Points** | 데모가 너무 느리면 지루함, 기술 용어가 많으면 이해 어려움, 접근성 미흡 시 사용 불편 |
| **기대** | 직관적인 UI, 빠르고 인상적인 데모, 결과물의 실용성 확인, 리포트 수령 |

### 사용 시나리오
1. 부스에 방문하여 데모 앱을 발견
2. 시작 페이지에서 프로젝트 아이디어 입력 (또는 예시 선택)
3. Kiro IDE 환경에서 AI-DLC 과정이 자동 진행되는 것을 관람
4. 마우스 포인터가 파일을 클릭하고, 채팅에 타이핑하고, 코드가 생성되는 과정을 시각적으로 체험
5. 결과 페이지에서 MVP UI, AWS 아키텍처, 프로젝트 예상 정보 확인
6. QR 코드 스캔 또는 이메일로 리포트 수령
7. 처음으로 돌아가기

## Persona 2: 부스 운영 스태프 (Booth Operator)

| 항목 | 내용 |
|------|------|
| **이름** | 이준호 (대표 페르소나) |
| **역할** | AWS Summit 부스 현장 운영 스태프 |
| **배경** | AI-DLC 팀 소속, 부스에서 방문객 안내 및 데모 지원 |
| **목표** | 데모 진행 상황을 실시간 모니터링하고, 방문객 관심사를 파악하여 부스 운영 최적화 |
| **동기** | 부스 운영 효율화, 인기 시나리오 파악, 문제 발생 시 빠른 대응 |
| **기술 수준** | 중급~고급 |
| **사용 환경** | 부스 내 별도 모니터/노트북에서 관리자 대시보드 접속 |
| **언어** | 한국어 |
| **Pain Points** | 데모 오류 발생 시 빠른 파악 어려움, 방문객 통계를 실시간으로 볼 수 없음 |
| **기대** | 실시간 세션 모니터링, 완료율 확인, 인기 아이디어 파악 |

### 사용 시나리오
1. 부스 오픈 전 관리자 대시보드 접속
2. 실시간으로 총 세션 수, 완료율, 평균 소요 시간 모니터링
3. 인기 프로젝트 아이디어 목록 확인하여 방문객 관심사 파악
4. 산업별 분포 차트로 트렌드 분석
5. 문제 발생 시 최근 세션 로그에서 원인 파악

## Persona 3: 마케팅 분석가 (Marketing Analyst)

| 항목 | 내용 |
|------|------|
| **이름** | 박지민 (대표 페르소나) |
| **역할** | AI-DLC 팀 마케팅 담당자 |
| **배경** | AWS Summit 이벤트 후 데이터 분석 및 보고서 작성 담당 |
| **목표** | Summit 이후 방문객 데이터를 분석하여 AI-DLC 마케팅 전략 수립에 활용 |
| **동기** | 방문객 관심 분야 파악, 데모 효과 측정, 후속 마케팅 캠페인 기획 |
| **기술 수준** | 초급~중급 |
| **사용 환경** | 사무실 데스크톱, 이벤트 종료 후 접속 |
| **언어** | 한국어 |
| **Pain Points** | 데이터가 체계적으로 정리되지 않으면 분석 어려움, 시각화 없으면 보고서 작성 힘듦 |
| **기대** | 세션 추이 차트, 산업별 분포, 완료율 통계, 데이터 내보내기 |

### 사용 시나리오
1. Summit 종료 후 관리자 대시보드 접속
2. 전체 기간 통계 확인 (총 방문자, 완료율, 평균 시간)
3. 산업별 관심 분포 차트 분석
4. 인기 프로젝트 아이디어 목록으로 트렌드 파악
5. 분석 결과를 마케팅 보고서에 활용
